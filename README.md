# מחשבון תוכניות חשמל – Electricity Plan Calculator

כלי קטן שעוזר לך למצוא את **תוכנית החשמל הטובה ביותר עבורך**, על בסיס נתוני הצריכה האישיים שלך מקובץ ה‑CSV של חברת החשמל.

הבהרה: לא מופיעות תוכניות שבהן לא ניתנת הנחה בפועל בחשבון אלא הטבה כלשהי (פזגז), ובתוכניות בהן ישנה הנחה מדורגת ו/או מנויים, צויין המחיר ללקוח שנה ראשונה/לא מנוי.

---

## איך זה עובד?

1. נכנסים לאתר חברת החשמל בכתובת https://www.iec.co.il/consumption-info-menu/remote-reading-info ושולחים לדוא"ל את קובץ ה‑CSV של צריכת החשמל שלך.
2. נכנסים לכלי בכתובת: https://tzagim.github.io/Electricity-Plan-Calculator
3. מעלים את הקובץ לכלי.  
4. הכלי מנתח את הצריכה שלך ומשווה בין תוכניות שונות.  
5. הכלי יציג המלצה על התוכנית המשתלמת ביותר עבורך.
